## Cómo descargar e instalar el proyecto

### 1- Clonar el repo
- git clone https://github.com/Tobias-Yegro/unq-ui-tobias-agustin-yegro-trabajo-final.git

### 2- Entrar en la carpeta del proyecto
- cd unq-ui-tobias-agustin-yegro-trabajo-final

### 3- Instalar dependencias: 
- npm install


## Cómo ejecutar el proyecto localmente

### 1- Ejecutar el servidor en modo desarrollo
- npm run dev

### 2- Abrir en el navegador la URL
- http://localhost:5173
